# Concurrent Programming

## Description
A collection of various mutlti-threaded programming projects created with Scala using the Scala Concurrency Library, written for the University of Oxford's Part A Concurrent Programming course. 


## Included Files
- Dining Philosphers: 3 variants to solve the dining philosophers problem concurrently, with logged files showing tests
- Sorting Networks: Using comparators to create a concurrent insertion sort network.
- AdaptiveQuad: A concurrent integral estimation using the bag-of-tasks with replacement paradigm.
- CheckDups: A code that checks for duplicates within two lists a and b
- ClientServer: A code that uses the client-server architecture to pair off men and women and tests it using FOL formulae.
- MatMul: Concurrent programming using the farm pattern to multiply two nxn matrices
- Merge: Merges two channels that are sorted and never close.
- Multiplex: A concurrent tagger-detagger algorithm
- PipeSort: A concurrent sorting network that "pipes" n inputs through n components
